    Table¶

    snowmobile.core.table

    Warning: This pge is incomplete

    snowmobile.Table is a data loading solution that implements the Bulk Loading from a Local File System to load a DataFrame into a table; an instance can be instantiated from defaults with a table (name str) and a df (DataFrame).

    Exhaustive documentation on its keyword arguments can be found in the API docs, with additional detail on the most broadly applicable included below.

    Note

    Alternate default values for the keyword arguments documented below can be specified in the default-table-kwargs section of snowmobile.toml

    Examples¶

    Setup¶

    In the following examples, the variable df is reserved for the df created with:

    """Generates reproducible DataFrame for snowmobile.Table examples"""
import datetime
import pandas as pd
import numpy as np

from typing import List

def rnp(c: List[str], seed: int = 1) -> str:
    """Returns a randomly chosen name within a list, c."""
    np.random.seed(seed)
    return np.random.choice(c)


data = {
    'col1': [rnp(['spam', 'eggs', 'ham']) for _ in range(24)],
    'col2': [rnp(['alpha', 'beta', 'gamma']) for _ in range(24)],
    'col3': [i for i in range(24)],
}

df = pd.DataFrame(data=data)
# print(df.shape)  #> (24, 3)
df.head(4)

# -- complete example; should run 'as is' --
    col1 col2 col3
    0 eggs beta 0
    1 eggs beta 1
    2 eggs beta 2
    3 eggs beta 3

    DataFrame to Existing Table (valid for load)¶

    Let's try it again but tell it to yell at us if the table already exists by providing if_exists='fail':

    from snowmobile.core.errors import ExistingTableError

try:
    _ = snowmobile.Table(
        df=df,
        sn=sn,
        table='sample_table',
        as_is=True,
        if_exists='fail',
    )
except ExistingTableError as e:
    print(e)

    `SAMPLE_TABLE` already exists and if_exists='fail' was provided;
'replace', 'append', or 'truncate' required to continue load process with a pre-existing table.
